Linux LVM逻辑卷的创建

本文档介绍了如何在 CentOS 7.9 系统中，将 sdb、sdc 和 sdd 三块磁盘通过 LVM 合并成一个逻辑卷，并挂载到 /data_v2 目录，同时确保文件系统格式与当前系统主分区一致，避免出现不必要的问题。

1. 查看当前磁盘情况

首先，查看当前磁盘和分区信息，确保磁盘是未分区状态，或者已经做好了数据备份。

2. 检查当前系统主分区文件系统格式

在创建文件系统之前，最好先检查当前系统主分区（如 / 或 /boot）的文件系统格式，以确保新创建的逻辑卷格式一致，避免不兼容问题。

3. 创建物理卷（PV）

将 sdb、sdc 和 sdd 三块磁盘创建为物理卷（PV）。

4. 创建卷组（VG）

将这三块物理卷合并为一个卷组，命名为 vg_data。

5. 创建逻辑卷（LV）

在 vg_data 卷组内创建一个逻辑卷，命名为 lv_data，并指定使用卷组中的全部剩余空间。

6. 查看卷组和逻辑卷

确认卷组和逻辑卷创建成功。

7. 格式化逻辑卷

对创建的逻辑卷进行格式化。确保格式化为与你当前系统主分区文件系统相同的格式（例如 xfs 或 ext4）。
